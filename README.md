# Indoor-Navigation-based-on-CNN
Independent Study

It is an application which enables user to enter destination by voic recognition, and then turn on the camera, it will recognize the user's current location.

We collected the indoor pictures by filming videos and used OpenCV to caputure images from video.

Next, pre-process the train dataset to grayscale, and start training.

Working Environment:

python3.7

Keras
